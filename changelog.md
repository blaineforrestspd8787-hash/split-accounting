# Changelog

## [1.1.0] — 2026-07-15

### 新增：其他分成

- 新增「其他分成」模式，独立于正常分成
- 固定比例：收益 = 总额 × 60%，成本 = 总额 × 40%
- 首页记账 Tab：分段控件切换「正常分成」/「其他分成」
- 记录 Tab：分段控件切换查看两种记录
- 其他分成独立存储、独立统计，不与正常分成混合
- 其他分成不支持 CSV 导出，比例固定不可在设置页配置

## [1.0.0] — 初始版本

- 每日收入录入，实时计算利润与分成
- 本月 / 今年统计
- 历史记录按月份筛选，支持编辑和删除
- CSV 导出
- 自定义分成比例（0~100%），保存后自动重算
- localStorage 本地存储 + Supabase 云端同步
- JSON 备份导入/导出
- 登录/注册（账本名 + 密码）
